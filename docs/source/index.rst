欢迎使用 LeaderDeep 深度学习软件！
===================================

**LeaderDeep** 是基于百度 PaddlePaddle 开发的深度学习软件。

集图像数据管理，标注，训练，部署于一体的全流程开发软件。

无论是否具备 AI 基础，极易理解，最快5分钟即可上手学会，15分钟完成模型训练。

.. 注意::

   文档更新或有不及时，请以实际软件使用为准。
